Raíz Quadrada - Implementação do método de Newton para o cálculo da raíz quadrada.

A forma mais comum utilizada pelos computadores para calcular raiz quadrada é o método de Newton. Tal método diz que se tivermos um palpite Y do valor da raíz de um número X, podemos efetuar uma manipulação simples para obter um palpite melhor (mais perto do valor real da raíz quadrada) calculando a média entre Y e X/Y. Por exemplo, podemos calcular a raíz quadrada de 2 como mostrado a seguir. Suponha que nosso valor inicial é 1:

Palpite Quociente Média
1 2/1 = 2 (2 + 1)/2 = 1.5
1.5 2/1.5 = 1.3333 (1.3333 + 1.5)/2 = 1.4167
1.4167 2/1.4167 = 1.4117 (1.4117 + 1.4167)/2 = 1.4142

Raíz Cúbica - Implementação do método de Newton para o cálculo da raíz cúbica.

O método de Newton para a raíz cúbica é semelhante ao da raíz quadrada mostrado acima, e baseia-se no fato de que se Y é uma aproximação da raíz cúbica de X, então uma melhor aproximação é dada pelo valor de (X/Y2 + 2Y)/3. Suponha que a aproximação inicial é 1, vamos calcular a raíz de 27:

Palpite Novo Valor
1 (27/1*1 + 2*1)/3 = 9.6666
9.6666 (27/9.6666*9.6666 + 2*9.6666)/3 = 6.5407
6.5407 (27/6.5407*6.5407 + 2*6.5407)/3 = 4.5709
4.5709 (27/4.5709*4.5709 + 2*4.5709)/3 = 3.4780
3.4780 (27/3.4780*3.4780 + 2*3.4780)/3 = 3.0626
3.0626 (27/3.0626*3.0626 + 2*3.0626)/3 = 3.0012


Fontes: [Java]


Algoritmo de Euclides - O Algoritmo de Euclides ou Algoritmo Euclidiano é um método matemático para a determinação do Máximo Divisor Comum (MDC), ele é bem simples e pode ser implementado muito facilmente tanto iterativamente quanto recursivamente. O procedimento MDC abaixo implementa o algoritmo de Euclides.

Sejam A e B dois números naturais diferentes de Zero e MDC(A, B) o procedimento que calcula seu Máximo Divisor Comum, temos: 
procedimento MDC(A, B):
    se A < B então
        troca(A, B) //troca  os valores de A e B 
    fim se.
    enquanto B != 0 faça
       R := A resto B // Armazena em R o resto da divisão de A por B
       A := B
       B := R
     fim enquanto
     retorne A
fim procedimento
